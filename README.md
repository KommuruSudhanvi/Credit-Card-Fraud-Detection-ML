# Credit Card Fraud Detection using Machine Learning

## Overview

This project uses machine learning techniques to classify credit card transactions as fraudulent or genuine.

## Dataset

The dataset contains 284,807 transactions with anonymized features.

The target variable:
0 - Genuine transaction
1 - Fraud transaction

## Workflow

- Data Understanding
- Exploratory Data Analysis
- Data Preprocessing
- Model Training
- Model Evaluation

## Models Used

- Logistic Regression
- Random Forest

## Evaluation Metrics

- Precision
- Recall
- F1 Score
- Confusion Matrix
